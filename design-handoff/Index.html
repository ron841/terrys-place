For the Engineer · Read These First

  1. Open HANDOFF.md — 60-second onramp, stack notes, audit of the live site.
  2. Read SLOTS.md — every dynamic slot in the design with its data shape, source, and fallback.
  3. Skim CONTENT-INVENTORY.md — which photos exist, which are placeholder, who provides what.
  4. Open Components.html — the visual + class-name reference.
  5. Open Home.html + Menu.html + Inner Pages.html — the comps to build against.
  6. Lift tokens from css/terrys.css — already structured for use as the theme's base CSS.
